# 🎓 同窓会参加者分析ダッシュボード

同窓会の参加者データを視覚的に分析するためのインタラクティブなWebダッシュボードです。
> **最終更新**: 2026年5月3日  
> **データ件数**: 170件  
> **データソース**: 同窓会0502.xlsx  
> 詳細な更新履歴は [CHANGELOG.md](CHANGELOG.md) をご覧ください。


## 📋 機能

### 主要機能
- **統計サマリー表示**: 総参加者数、オンライン決済率、ツアー参加率、バス利用率
- **インタラクティブなグラフ**: Chart.jsを使用した美しいグラフ表示
  - 支払方法の分布（円グラフ）
  - キャンパスツアー参加状況（円グラフ）
  - 送迎バス利用状況（円グラフ）
  - 卒業年別参加者数（棒グラフ）
  - 年齢別参加者数（棒グラフ）※卒業時19歳として計算
  - 人気クラブ TOP 10（横棒グラフ）
- **フィルタリング機能**: 複数条件でデータをフィルタリング
  - 卒業年・回
  - 会費支払方法
  - キャンパスツアー申込
  - 送迎バス利用
- **レスポンシブデザイン**: PC、タブレット、スマートフォンに対応

### 追加機能
- CSVエクスポート機能（コンソールから実行可能）
- 統計サマリー表示（コンソールから実行可能）

## 🚀 使い方

### 1. ファイル構成

```
reunion-dashboard/
├── index.html          # メインHTMLファイル
├── css/
│   └── style.css      # スタイルシート
├── js/
│   ├── main.js        # メインロジック
│   ├── dataLoader.js  # CSVデータ読み込み
│   ├── statistics.js  # 統計計算
│   ├── charts.js      # グラフ描画
│   └── filters.js     # フィルタリング機能
├── data/
│   └── participants.csv # 参加者データ
└── README.md          # このファイル
```

### 2. ダッシュボードを開く

#### 方法1: ローカルサーバーを使用（推奨）

```bash
cd reunion-dashboard
python3 -m http.server 8000
```

ブラウザで `http://localhost:8000` を開く

#### 方法2: 直接HTMLファイルを開く

`index.html` をダブルクリックしてブラウザで開く

**注意**: セキュリティ制限により、一部のブラウザではCSVファイルの読み込みに失敗する場合があります。その場合は方法1を使用してください。

### 3. フィルターの使い方

1. **卒業年代でフィルター**: ドロップダウンから特定の卒業年・回を選択
2. **支払方法でフィルター**: オンライン決済、銀行振込等を選択
3. **キャンパスツアーでフィルター**: 参加する/参加しないを選択
4. **送迎バスでフィルター**: 利用する/利用しないを選択
5. **フィルターをリセット**: 「フィルターをリセット」ボタンをクリック

フィルターを変更すると、すべてのグラフとテーブルが自動的に更新されます。

### 4. コンソールコマンド

ブラウザの開発者ツール（F12）のコンソールで以下のコマンドが使用できます：

```javascript
// フィルター済みデータをCSVでエクスポート
exportToCSV()

// 統計サマリーをコンソールに表示
showStatisticsSummary()

// すべてのフィルターをリセット
resetAllFilters()
```

## 📊 データ形式

CSVファイルは以下の列を含む必要があります：

- **回答**: 参加状況（「参加」のみ表示）
- **卒業年・回**: 卒業年度と回数
- **所属クラブ名**: 所属していたクラブ名
- **会費支払方法**: オンライン決済、銀行振込等
- **キャンパスツアー申込**: 参加する/参加しない
- **送迎バス**: 利用する/利用しない
- **登録日時**: 登録日時

## 🐛 トラブルシューティング

### CSVファイルが読み込めない

**原因**: ブラウザのセキュリティ制限（CORS）

**解決策**: ローカルサーバーを使用してください

```bash
python3 -m http.server 8000
```

### グラフが表示されない

**原因**: Chart.jsの読み込み失敗

**解決策**: インターネット接続を確認してください（CDN経由で読み込んでいます）

### データが正しく表示されない

**原因**: CSVファイルのエンコーディングまたは形式の問題

**解決策**: 
1. CSVファイルがUTF-8エンコーディングであることを確認
2. 列名が正しいことを確認
3. ブラウザのコンソール（F12）でエラーメッセージを確認
